== Short Description ==
A lightweight plugin to add a customizable eco-friendly badge to your WordPress site.

== Description ==
Eco-Friendly Website Badge is a lightweight plugin that adds a customizable badge to your WordPress site, showcasing your commitment to sustainability. The badge reads "Powered by Ecozaar â€“ Sustainable Web" and links directly to Ecozaar.in.

== Features ==
* Customizable badge with placement options: footer, sidebar, or via shortcode.
* Selectable badge sizes: small, medium, large.
* Easy-to-use settings page in the WordPress admin.
* Secure code with proper sanitization and escaping following WordPress best practices.

== Installation ==
1. Upload the `ecozaar-badge` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings under **Settings > Eco Badge**.
4. If using the shortcode placement, insert the badge anywhere using the `[ecozaar_badge]` shortcode.

== Frequently Asked Questions ==
= How do I change the badge placement? =
You can select the placement (footer, sidebar, or shortcode) from the plugin settings page in your WordPress admin.

= How do I adjust the badge size? =
Choose your preferred badge size (small, medium, large) from the settings page.
